Quick Tips

  • Tap this icon to go to the Global menu and access all of the record types in Saleslogix Mobile: search using SpeedSearch (Saleslogix 8.0 and higher), find Help, configure settings, and log out.
  • Tap this icon to go to the Contextual (right) menu to access available hashtags and Key Performance Indicators (KPIs). This is available on Saleslogix 8.0 and higher.
  • The configure button on the Global menu allows you to choose the types of records to show in the "Go To" section and specify the order in which they appear.
  • This icon indicates that you can expand and collapse panels to show/hide additional details.
  • This icon indicates that you can add a new record to Saleslogix. It is available on all list views.
  • Underlined text in detail views signifies a link action (such as browsing to a URL or jumping to an Account).
  • The "Clear Storage" option under Settings is strictly a trouble-shooting device to clear locally stored configuration.

Quick Actions

  • Following phone call or email actions, a view enabling you to type notes and save to history displays.
  • While in the list view, click the record icon to see and access short-cuts to frequently-used features. (Some quick action icons may be unavailable. For example, in My Activities, the Decline action will be unavailable if you have already accepted the meeting.) My Activities and Attachments are available in Saleslogix 8.0 and higher.
  • The "View address" action takes you to the native map application on supported devices.

Search

  • There are multiple ways to search:

    • From the Global menu, find all related accounts, contacts, leads, opportunities, activities, notes\history, and tickets. Enter more text to improve the accuracy of your search.
    • From the Contextual menu, filter list view results using pre-defined hashtag keywords and\or the fields specified below.
    • For records such as Products, filter list view results from the lookup bar using the field specified below.
