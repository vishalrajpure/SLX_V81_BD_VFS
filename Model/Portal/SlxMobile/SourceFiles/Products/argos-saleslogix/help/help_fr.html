Conseils rapides

  • Appuyez sur cette icône pour ouvrir le menu Global et accéder à tous les types de fiches de Saleslogix Mobile : effectuer une recherche avec SpeedSearch, obtenir de l'aide, configurer des paramètres et se déconnecter.
  • Appuyez sur cette icône pour ouvrir le menu contextuel (clic droit) et accéder aux mots clés et aux indicateurs de performance clés (KPI) disponibles.
  • Le bouton de configuration du menu Global vous permet de choisir les types de fiches à afficher dans la section Aller à et de spécifier leur ordre d'affichage.
  • Cette icône indique que vous pouvez développer et réduire les volets afin d'afficher ou de masquer les détails.
  • Cette icône indique que vous pouvez ajouter une fiche dans Saleslogix. Elle est disponible dans toutes les vues Liste.
  • Le texte souligné dans les vues Détail indique une action hypertexte (par exemple, l'accès à une URL ou à un compte).
  • L'option Effacer le stockage de la section Paramètres est un simple outil de dépannage permettant d'effacer la configuration stockée en local.

Actions rapides

  • Après un appel téléphonique ou un e-mail, une vue vous permettant de saisir des notes et de les enregistrer dans l'historique s'affiche.
  • En mode Liste, cliquez sur l'icône de la fiche pour accéder aux raccourcis des fonctions fréquentes. (Certaines icônes d'actions rapides peuvent ne pas être disponibles. Par exemple, dans Mes activités, l'action Refuser ne sera pas disponible si vous avez déjà accepté la réunion.)
  • L'action d'affichage de l'adresse vous permet d'accéder à l'application de mappage native sur les appareils pris en charge.

Rechercher

  • La recherche peut s'effectuer de différentes manières :

    • Dans le menu Global, vous pouvez effectuer une recherche sur tous les comptes associés, contacts, prospects, opportunités, activités, notes/historique et tickets. Saisissez davantage d'informations pour affiner votre recherche.
    • Dans le menu contextuel, vous pouvez filtrer les résultats de la vue Liste avec des mots clés prédéfinis et/ou les champs spécifiés ci-après.
    • Pour les fiches telles que Produits, filtrez les résultats de la vue Liste depuis la barre de recherche à l'aide du champ spécifié ci-après.
