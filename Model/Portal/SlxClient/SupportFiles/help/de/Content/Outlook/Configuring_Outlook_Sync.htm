Konfigurieren von Outlook Sync und Outlook Integration

Synchronisierungs- und Integrationseinstellungen sollten in Microsoft Outlook konfiguriert werden.

 Die Optionen "Verbindung" und "Mit SLX senden/In Historie aufzeichnen" sind evtl. bereits für Web Client-Benutzer konfiguriert, die ihre Optionen im Desktop Manager festlegen.

Für unterschiedliche Datensatztypen können verschiedene Einstellungen konfiguriert werden. Sie können beispielsweise die Synchronisierung in eine Richtung für Kontakte und die Synchronisierung in beide Richtungen für Kalenderelemente aktivieren. In den folgenden Schritten werden die empfohlenen Synchronisierungseinstellungen zwischen Microsoft Outlook und Saleslogix beschrieben.

Bevor Sie Outlook konfigurieren können:

  • Microsoft Outlook muss installiert sein und auf dem lokalen Rechner ausgeführt werden.
    • Web – Die Desktop-Integration wird nur dann installiert, wenn Microsoft Office, einschließlich Outlook, bereits installiert ist.
    • Windows – Falls Microsoft Outlook nicht bereits installiert war, als der Saleslogix Client installiert wurde, müssen Sie weitere Konfigurationsschritte ausführen.
  • Sie müssen den Verbindungsserver und die Portnummer sowie Ihren Benutzernamen und Ihr Passwort kennen.
    Weitere Informationen erhalten Sie von Ihrem Systemadministrator.
  • Sie müssen eine Kontaktsynchronisierungsgruppe für die Kontakte erstellen, die Sie zwischen Outlook und Saleslogix synchronisieren möchten.

So gehen Sie zum Konfigurieren vor

  1. Öffnen Sie Microsoft Outlook.
  2. Falls Sie keine Eingabeaufforderung erhalten, klicken Sie auf die Schaltfläche Optionen.
    Wenn die Schaltfläche "Optionen" nicht erscheint, wurde Outlook-Synchronisierung nicht korrekt installiert.
  3. Verwenden Sie die Registerkarten, um die folgenden Optionen festzulegen:
    • Verbindung – Verbindung mit dem Saleslogix-Server. Ist erforderlich für die Kontakt und Kalender-Synchronisierung und E-Mail-Integration.
    • Sync – Synchronisierungsoptionen für Kontakte, Termine (Anrufe, Meetings und persönliche Aktivitäten) und Aufgaben. Ist erforderlich für Kontakt und Kalender-Synchronisierung .
    • Senden SLX An Historien aufzeichnen – Optionen für Eingabeaufforderungen, zum Anzeigen von Informationen und für Anlagenverhalten, wenn "Senden SLX" oder "An Historie aufzeichnen" verwendet wird. Wird verwendet für E-Mail-Integration.
    • Adressbuch – Anzeigeoptionen für Ihr Saleslogix Adressbuch. Wird verwendet für E-Mail-Integration.
    • Protokollierung – Möglichkeit, die Protokollierung ein- und auszuschalten und Protokollierungsinformationen zur Fehlerbehebung anzuzeigen. Wird verwendet für Kontakt- und Kalender-Synchronisierung
    • Alle Optionen – Möglichkeit, Optionen als Werte-Paare anzuzeigen. Wird nur für fortgeschrittene Benutzer empfohlen.
       

  4. Klicken Sie auf "OK".
