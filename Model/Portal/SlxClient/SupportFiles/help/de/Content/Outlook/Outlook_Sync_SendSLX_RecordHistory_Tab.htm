Festlegen der Optionen "Senden SLX An Historien aufzeichnen in Saleslogix Connector

Verwenden Sie die Option "Senden SLX An Historien aufzeichnen", um Optionen für die E-Mail-Historie und Anlagen festzulegen.
  • Diese Registerkarte ist nur für Outlook Integration erforderlich.
  • Für Web Client-Benutzer, die ihre Optionen im Desktop Manager festlegen, sind diese Optionen evtl. bereits konfiguriert.

So legen Sie Optionen fest

  1. Click the Saleslogix Options button, und klicken Sie auf Senden SLX An Historien aufzeichnen.
  2. Wählen Sie aus, wann Sie zu einer Aktion aufgefordert werden möchten, wenn eine Outlook-E-Mail als Saleslogix-Historiendatensatz protokolliert wird.
    1.  Nachfrage bei doppelten Kontakten – Fragt nach, wenn ein doppelter Kontakt oder Interessent gefunden wird.
    2.  Nachfrage bei Kontakten – Fordert Sie zur Auswahl auf, mit welchen Kontakten oder Interessenten der Historieneintrag verknüpft werden soll.
    3. Nachfrage bei unbekannten Kontakten – Fragt nach, wenn ein Kontakt oder Interessent nicht in der Datenbank gefunden wird.
    4. Historien vor dem Speichern anzeigen – Blendet das Dialogfeld "E-Mail abschließen" ein, bevor eine E-Mail-Nachricht in der Historie aufgezeichnet wird.
    5.  Tray Benachrichtigungen anzeigen – Zeigt Einblendmeldungen an, in denen die jeweiligen Desktop Manager-Aktivitäten beschrieben werden.
  3. Legen Sie die Anlagenoptionen fest.
    1. Wählen Sie die Anlagen bis kB... igorieren.
      Verwenden Sie diese Option, um kleine Dateianlagen, z. B. E-Mail-Signaturen, zu ignorieren, damit sie nicht als Saleslogix-Anlage gespeichert werden.
    2. Listen Sie im Feld Anlagen mit folgenden Dateinamen ignorieren die Dateierweiterungen oder Dateitypen auf, die nicht als Saleslogix-Anlagen gespeichert werden sollen.
  4. Klicken Sie auf OK, wenn Sie alle Optionen festgelegt haben, oder wählen Sie eine andere Optionsregisterkarte aus.
