Festlegen der Adressbuch-Optionen für Saleslogix Connector

Verwenden Sie die Adressbuch-Optionen, um das Aussehen Ihres Saleslogix-Adressbuches einzustellen.

Diese Registerkarte ist nur für Outlook Integration erforderlich.

So legen Sie Optionen fest

  1. Click the Saleslogix Options button, und klicken Sie bei Bedarf auf Adressbuch.
  2. Wählen Sie unter Kontaktnamen anzeigen als aus, wie die Namen in Ihrem Adressbuch erscheinen sollen.
  3. Wählen Sie unter Beim Anzeigen von Adressbuchs die Optionen zur benutzerspezifischen Anpassung Ihres Adressbuchs aus, oder heben Sie die Markierungen dieser Optionen auf.
  4. Klicken Sie auf OK, wenn Sie alle Optionen festgelegt haben, oder wählen Sie eine andere Optionsregisterkarte aus.
