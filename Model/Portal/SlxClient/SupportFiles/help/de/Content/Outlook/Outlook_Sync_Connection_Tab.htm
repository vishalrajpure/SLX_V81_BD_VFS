Festlegen der Verbindungsoptionen für Saleslogix Connector

Verwenden Sie die Verbindungsoptionen, um Ihre Verbindung mit dem Saleslogix-Server einzurichten.
  • Diese Registerkarte ist sowohl für Outlook-Synchronisierung als auch für Outlook Integration erforderlich.
  • Für Web Client-Benutzer, die ihre Optionen im Desktop Manager festlegen, sind diese Optionen evtl. bereits konfiguriert.

So legen Sie Optionen fest

  1. Click the Saleslogix Options button, und klicken Sie ggf. auf Verbindung.
  2. Geben Sie im Feld Server Ihren Servernamen ein.
    • Saleslogix Client und Web Client-Benutzer – Geben Sie die URL ein, die Ihr Administrator für die Verbindung mit Saleslogix bereitgestellt hat. Verwenden Sie das folgende Format:
      http://servername:port or https://servername:port
    • Offline Web Client-Benutzer – Geben Sie den URL ein, um eine Verbindung mit Ihrem lokalen Saleslogix-Server herzustellen. Verwenden Sie das folgende Format:
      http://localhost:port
      Der Port ist gewöhnlich 8088.

  3. Führen Sie einen der folgenden Schritte aus:

    • Wenn Sie sich mithilfe Ihrer Windows-Kontoinformationen bei Saleslogix anmelden, wählen Sie die Option Windows Authentifizierung verwenden.
    • Geben Sie Ihren Saleslogix-Benutzername und Ihr Passwort ein.
  4. Klicken Sie auf Test.
    Wenn der Test fehlschlägt, überprüfen Sie Ihre Verbindungsinformationen. Wenn die Informationen richtig sind und Sie dennoch keine Verbindung herstellen können, wenden Sie sich an Ihren Administrator.
  5. Klicken Sie auf OK, wenn Sie alle Optionen festgelegt haben, oder wählen Sie eine andere Optionsregisterkarte aus.
