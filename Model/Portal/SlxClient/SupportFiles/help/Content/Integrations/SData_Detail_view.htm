Using the SData Sync Configuration View

Use the SData Sync Configuration view to set up the Saleslogix feed used to synchronize Saleslogix with integrated systems.

To configure SData

  1. On the Navigation Bar, click SData Sync Configuration
  2. Set up the Saleslogix Feed. Use the following boxes to configure the Saleslogix Feed. The feed will automatically update as you enter the data.
    1. If your SData URL uses Secure Socket Layer, select the Use SSL (Secure Socket Layer) check box.
    2. In the Server box, type the name of the server for your SData URL.
    3. In the Port box, type the port number for your SData URL.
  3. In the User Name box, leave as "ADMIN" or type the user name for the Saleslogix database.

    Windows Authentication must be configured for the administrative user. For detailed steps, see the "Using Windows Authentication" topic in the Administrator Help

  4. In the Password box, type the password for the user name.
  5. In the Display Name box, leave as" Saleslogix" or type the display name you want to represent the SData feed.
  6. Click the Test Link button and view the information in the Status box.
    If the feed did not test successfully, review the information boxes, make any changes, and test the link again.
  7. Click Save.

Detail view tabs

The lower section contains the SData Sync Configuration view tabs. Click a link to see more information about each tab.
