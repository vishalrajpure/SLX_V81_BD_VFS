Adding and Editing Report Conditions

Report conditions enable you to modify the content of a pre-defined report to generate the most useful data. The options available to you will vary depending on the selections that you make.

Match By  
Date Range

From the pick list, select one of the default time periods, or use Specific Dates to select a custom time period.

Group

Groups are a collection of records. You can use groups to work with a subset of records.

Use the Lookup to select from a list of groups.

Query

From the pick lists, select the following:

  • Table,
  • Field
  • Operator
  • Value: Either type in a value or select a date.

For example.

Table: Opportunity

Field: CREATEDATE

Operator: Is After

Date:  October 1.
User

Use the Lookup to select from a list of users.
