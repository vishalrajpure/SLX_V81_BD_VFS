Associating or Disassociating with a Contact

If your Saleslogix implementation includes creating a contact for each Saleslogix user, you can easily associate the user and contact from the Users List view.

The system administrator and users with the appropriate role can access these features.

How Do I?

Associate a user with a contact
  1. On the Navigation Bar, click Administration, and then click Users.
  2. Select the user you want to associate with a contact.
  3. In the Task Pane, under User Tasks, click Associate with Contact.
  4. In the Associate User with Contact dialog box, click the Contact Find button.
  5. Use the Lookup to find the contact you want to associate with the user, and then click OK
  6. Click OK.
Disassociate a user from a contact
  1. On the Navigation Bar, click Administration, and then click Users.
  2. Select one or more users you want to disassociate from any contacts.
  3. In the Task Pane, under User Tasks, click Disassociate from Contact.
  4. Click OK.
