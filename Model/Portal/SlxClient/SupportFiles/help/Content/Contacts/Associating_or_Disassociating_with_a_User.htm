Associating or Disassociating with a User

If your Saleslogix implementation includes creating a contact for each Saleslogix user, you can easily associate the user and contact from the Contacts List view.

How Do I?

Associate a user with a contact
  1. On the Navigation Bar, click Contacts.
  2. Select the contact you want to associate with a user.
  3. In the Task Pane, under Contact Tasks, click Associate with User.
  4. In the Associate Contact with User dialog box, click the User Find button.
  5. Use the Lookup to find the user you want to associate with the contact, and then click OK
  6. Click OK.
Disassociate a user from a contact
  1. On the Navigation Bar, click Contacts.
  2. Select one or more contacts you want to disassociate from any users.
  3. In the Task Pane, under Contact Tasks, click Disassociate from User.
  4. Click OK.
