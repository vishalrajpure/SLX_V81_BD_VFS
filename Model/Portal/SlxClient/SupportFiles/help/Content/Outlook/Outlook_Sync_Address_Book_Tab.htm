Setting Saleslogix Connector Address Book Options

Use the Address Book options to set up your Saleslogix address book appearance.

This tab is required for Outlook Integration only.

To set options

  1. Click the Saleslogix Options button, and, if necessary, click Address Book.
  2. Under Show contact names as, select how you want names to appear in your address book.
  3. Under When showing Address Book, select or clear the options to customize your address book.
  4. Click OK if you are finished setting all of your options, or select another options tab.
