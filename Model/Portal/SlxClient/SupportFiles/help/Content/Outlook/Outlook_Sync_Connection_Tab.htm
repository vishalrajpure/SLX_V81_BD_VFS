Setting Saleslogix Connector Connection Options

Use the Connection options to set up your connection to the Saleslogix server.

  • This tab is required for both Outlook sync and Outlook Integration.
  • These options may already be configured for Web Client users who set their options in the Desktop Manager.

To set options

  1. Click the Saleslogix Options button and, if necessary, click Connection.
  2. In the Server box, type your server name.
    • Saleslogix Client and Web Client users - Type the URL that your administrator provided in order to connect to Saleslogix. Use the following format:
      http://servername:port or https://servername:port
    • Offline Web Client users - Type the URL to connect to your local Saleslogix server. Use the following format:
      http://localhost:port
      The port is usually 8088.

  3. Do one of the following:

    • If you use your Windows account information to log in to Saleslogix, select the Use Windows Authentication option.
    • Enter your Saleslogix User Name and Password.
  4. Click Test.
    If the test fails, review your connection information. If the information is correct and you still cannot connect, contact your administrator.
  5. Click OK if you are finished setting all of your options, or select another options tab.
