Définition des options de connexion du connecteur Saleslogix

Utilisez les options de connexion pour configurer votre connexion au serveur Saleslogix.
  • Cet onglet est requis à la fois pour Synchronisation de Outlook et pour l'Intégration d'Outlook.
  • Il est possible que ces options soient déjà configurées pour les utilisateurs du client Web qui définissent les options dans le Gestionnaire de Bureau.

Pour définir les options

  1. Click the Saleslogix Options button et, si nécessaire, cliquez sur Connexion.
  2. Dans le champ Serveur, tapez votre nom de serveur.
    • Saleslogix Client et utilisateurs de Client Web - Tapez l'URL que votre administrateur a fourni pour vous connecter à Saleslogix. Utilisez le format suivant :
      http://servername:port ou https://servername:port
    • Utilisateurs du client Web hors ligne - Tapez l'URL pour vous connecter à votre serveur Saleslogix local. Utilisez le format suivant :
      http://localhost:port
      Le port est généralement 8088.

  3. Effectuez l'une des opérations suivantes :

    • Si vous utilisez les informations de votre compte Windows pour vous connecter à Saleslogix, sélectionnez l'option Utiliser le compte utilisateur Windows.
    • Saisissez votre Saleslogix Nom d'utilisateur et votre Mot de passe.
  4. Cliquez sur Tester.
    Si le test échoue, examinez vos informations de connexion. Si les informations sont correctes et que vous ne pouvez toujours pas vous connecter, contactez votre administrateur.
  5. Cliquez sur OK si vous avez fini de définir toutes vos options, ou sélectionnez un autre onglet Options.
