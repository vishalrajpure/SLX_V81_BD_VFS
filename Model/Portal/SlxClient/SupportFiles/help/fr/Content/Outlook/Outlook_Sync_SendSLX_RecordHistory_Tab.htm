Définition des options Envoyer SLX/Enregistrement dans l'historique du connecteur Saleslogix

Utilisez Envoyer SLX/Enregistrement dans l'historique pour définir les options d'historique des e-mails et de pièce jointe.
  • Cet onglet est requis pour l'Intégration d'Outlook seulement.
  • Il est possible que ces options soient déjà configurées pour les utilisateurs du client Web qui définissent les options dans le Gestionnaire de Bureau.

Pour définir les options

  1. Click the Saleslogix Options button et cliquez sur Envoyer à SLX/Enregistrement dans l'historique.
  2. Sélectionnez dans quel cas vous souhaitez recevoir une notification lorsque vous consignez un e-mail Outlook en tant qu'élément d'historique Saleslogix.
    1.  Afficher les contacts en double - Vous informe lorsqu'un contact ou un lead en double est détecté.
    2.  Afficher les contacts - Vous invite à sélectionner les contacts ou leads auxquels l'élément d'historique doit être associé.
    3. Afficher les contacts non résolus - Vous informe lorsqu'un contact ou un lead est introuvable dans la base de données.
    4. Afficher les détails de l'historique avant d'enregistrer - Vous affiche la boîte de dialogue complète de l'e-mail avant qu'un message e-mail ne soit enregistré dans l'historique.
    5.  Afficher les notifications de tiroir - Affiche des messages pop-up qui décrivent les activités du Gestionnaire de Bureau à mesure qu'elles se produisent.
  3. Définissez les options de pièce jointe.
    1. Sélectionnez la Taille minimum de pièce jointe à ignorer (Ko).
      Utilisez cette option pour ignorer les petits fichiers en pièce jointe, tels que les signatures d'e-mail, de sorte à ne pas les enregistrer comme des pièces jointes Saleslogix.
    2. Dans le champ Ignorer les pièces jointes avec ces noms de fichiers, listez les extensions de fichier ou tout type de fichier qui ne doit pas être enregistré comme une pièce jointe Saleslogix.
  4. Cliquez sur OK si vous avez fini de définir toutes vos options, ou sélectionnez un autre onglet Options.
