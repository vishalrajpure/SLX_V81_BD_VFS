Définition des options de journalisation du connecteur Saleslogix

Utilisez l'onglet Consignation pour activer la consignation ou afficher les informations de consignation en cas de problèmes de synchronisation ou si vous êtes invité à le faire par votre administrateur.

Cet onglet s'applique uniquement à Synchronisation de Outlook.

Pour définir les options

  1. Click the Saleslogix Options button, et, si nécessaire, cliquez sur Journalisation.
  2. Sélectionnez Activer la journalisation.
  3. Affichez les informations de consignation dans le champ Dernières erreurs.
  4. Cliquez sur OK si vous avez fini de définir toutes vos options, ou sélectionnez un autre onglet Options.
