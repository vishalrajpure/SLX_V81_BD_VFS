Form Properties

You can view and change the values for the following properties of this form:

Basic Property Description
Active Control The identifier of the control you want the user's cursor to start on when the page first displays. For example, AccountName on the InsertAccount form.
Name The name of the form, for example, AccountDetails. This information is read-only in the Form Designer.
Description The description of the form.
Basic Property Description
Use Entity Name as Title When set to true, the form name is determined by the type of entity the form is based on. For example, if the form is based on Account, the form title is the current account name.
Basic Property Description
Portal The name of the web site the view is part of. This information is read-only in the Form Designer.
View The name of a view that contains the form. For example, Account Details. This information is read-only in the Form Designer.
Modes For this view, the modes the form is rendered in: Detail, Insert, and List. This information is read-only in the Form Designer.
Description The description of the view. This information is read-only in the Form Designer.
